## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: quick-range controls minimum tap height `36px`; custom-date trigger, primary CTA, and other primary interactive controls minimum height `40px`

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 | 1.5 |
| Label | 14px | 600 | 1.5 |
| Heading | 20px | 600 | 1.2 |
| Display | 28px | 600 | 1.2 |

Rules:
- Use `28px` display only for overview card primary values and page-level numeric emphasis.
- Use `20px` heading for chart/card section titles and date-range headline blocks.
- Use `14px` labels for toolbar pills, legends, delta labels, and tooltip row labels.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#F7F8FA` | Page background and low-emphasis layout canvas |
| Secondary (30%) | `#FFFFFF` | Cards, toolbar container, tooltip surface, and elevated modules |
| Accent (10%) | `#2F6BFF` | Selected ranges, primary CTA, evaluated-series bars, and hover highlight |
| Destructive | `#D14343` | Worsening ratio deltas and error state treatment in this phase |

Accent reserved for: selected quick-range item, custom-date apply action, evaluated-case series in the trend chart, chart hover focus state

Additional color rules:
- Total outpatient volume bars use neutral gray, not accent, so evaluated cases remain the single emphasized quantity series.
- White tooltip surface uses a light gray border to preserve the locked PRD style from `02-CONTEXT.md`.
- Do not apply the accent color to every clickable element; secondary controls stay neutral unless selected.

## Interaction Contract

- Toolbar pattern: use quick-range chips plus a custom-date entry point in the same row; quick ranges are the primary path, custom date is secondary.
- Toolbar state is single-source: one selected range drives overview cards, trend chart, absolute date label, and chart grain. No module may hold its own divergent time state.
- Default load state: selected range is `近三个月`, cutoff is `昨日`, and the page renders 13 weekly buckets inherited from the Phase 1 semantic layer.
- Date presentation: show explicit start and end dates in the toolbar summary, never only the relative label.
- Overview card layout: render exactly five cards in this order: 门诊量, 评估病例数, 诊断依据不完整比例, 缺失诊断比例, 整体诊断质量指数.
- Card content hierarchy: primary value first, delta row second. Do not add tertiary descriptive copy inside the card.
- Delta rule: use `N/A` when prior-period value is zero, missing, or otherwise non-comparable. Use `较上期 --` when unchanged.
- Delta semantics: for quality problem ratios, upward change is red with `↑` and downward change is green with `↓`; for count metrics, increase is red with `+` and decrease is green with `-`.
- Trend chart structure: use a dual-axis mixed chart with left axis for the two ratio lines and right axis for total outpatient volume plus evaluated-case bars.
- Quantity bars: show total outpatient volume as the neutral base bar and evaluated cases as the blue overlay bar to preserve the locked assessed-vs-unassessed reading.
- Tooltip: axis-triggered, white surface, light border, and includes time label plus all four core metrics in one hover card.
- Empty state: when the selected window has no data, cards and chart both render empty-state treatment instead of zero values.
- Partial-data rule: if one metric is unavailable, preserve the rest of the module and mark only the missing metric as unavailable.
- Frontend semantic boundary: UI formats and arranges data only. It must not derive buckets, ratio math, or prior-period comparison outside Phase 1 outputs.
